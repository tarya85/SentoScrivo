TI SERVE UNA CONSULENZA DI TRASCRIZIONE?

INVIACI UN ESEMPIO DI FILE AUDIO/immagine PER AVERE UN PREVENTIVO DETTAGLIATO

Possiamo offrire un preventivo completo di tutte le specifiche, analizzando un esempio o l’intero file audio/immagine per servizi di verbalizzazione o trascrizione in formato digitale . I passaggi da seguire sono piuttosto semplici:

  1. Compilare il modulo di seguito inserendo tutti i campi obbligatori.
  2. Per l’invio del file audio o di immagini (ad esempio scansioni di documenti o screenshot di schermate dal cellulare/monitor) sarà necessario caricare i file su wetransfer e fornirci il link breve ricavato da wetransfer al termine del caricamento. Per caricare il file su wetransfer e aver il link breve sarà necessario seguire i passaggi dal 3 al 9.
  3. Andare sul sito https://wetransfer.com/
  4. Cliccare sui tre puntini di fianco al tasto “trasferisci”.
  5. All’apertura delle opzioni selezionare la voce “invia come link”.
  6. Caricare il file e attendere il completamento del caricamento.
  7. Alla conclusione copiare il link riportato da wetransfer.
  8. Incollare il link nel campo corrispondente nel nostro preventivatore.
  9. Inviare a Sentoscrivo la rcihiesta di preventivo specificando evenutali note nel campo di testo libero.

Sarete ricontattati il prima possibile da un nostro responsabile tramite email o numero di telefono, con un preventivo per il servizio richiesto.

COMPILA IL MODULO PER UN PREVENTIVO

Riceverai il preventivo dettagliato sulla base dell’analisi delle tue richieste, o verrai ricontattato da un nostro responsabile per maggiori chiarimenti e informazioni.

Le voci contrassegnate dall’asterisco * sono obbligatorie per la richiesta di preventivo.

Clicca sull’immagine di lato per capire come ricavare il link di wetransfer prima di caricare il file da inviare.
